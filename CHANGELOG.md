# Changelog

All notable changes to this project are documented here.

## 0.2.0 - 2026-07-15

- Add a dependency-free cross-platform installer with install, update, and uninstall actions.
- Add focused MCP and installer tests using Node.js built-in test tools.
- Add Windows, macOS, and Linux quality checks in GitHub Actions.
- Improve setup, verification, workflow, and security documentation.
- Preserve the lightweight local architecture, restricted permission modes, and credential-free design.

## 0.1.0 - 2026-07-12

- Initial public release.
